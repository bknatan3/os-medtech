# API MVP

## Auth
- `POST /auth/login`
- `GET /auth/me`

## Admin
- `GET /users`
- `POST /users`
- `GET /customers`
- `POST /customers`
- `GET /equipment`
- `POST /equipment`
- `GET /templates`
- `POST /templates`

## Work Orders
- `GET /work-orders`
- `POST /work-orders`
- `PATCH /work-orders/:id`
- `POST /work-orders/:id/status`

## Sync
- `POST /sync`

Swagger em `GET /api`.
